# Python Challenges

A collection of Python programming challenges and utilities.

## Project Structure

- `src/challenges/`: Algorithm implementations
- `src/utils/`: Utility functions and classes
- `tests/`: Test suite using pytest

## Development

### Setup

```bash
pip install -e ".[dev]"
```

### Run Tests

```bash
pytest
```

### Format Code

```bash
black src tests
isort src tests
```
